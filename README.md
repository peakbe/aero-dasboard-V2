# Dashboard Aéroportuaire

Affiche météo, sens de piste, avions confirmés et sonomètres autour des aéroports **EBLG** et **EBCI**.

## Installation

1. Renseigner les clés API dans `js/config.js`
2. Héberger sur GitHub Pages

## API utilisées

- Météo → exemple : OpenWeatherMap / MeteoFrance
- Aviation → FlightAware / ADS-B API
